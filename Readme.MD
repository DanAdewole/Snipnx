```python

```

### Recursive Factorial Function

```python
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)
```

Python recursive function to calculate factorial.

### Person Class

```python

```

### List Comprehension

```python

```

Python list comprehension to square numbers.
